Data analysis pipeline for investigation of RgzA as the regulator of zorbing

**"A Sensory Pathway Controls Motile Microcolony Formation and Spatial Organization in Gliding Bacteria"

## Contents
Codes to perform the analysis and generate the plots for each of the figures in this work are organzied in respective folders.

## Requirements

- Python 3.10+
- Jupyter Notebook
- Standard scientific stack: pandas, numpy, matplotlib, seaborn, skimage. scipy 

## How to use

1. Clone or download this repository
2. Open notebooks in Jupyter

## Data availability

- Raw RNA-seq data: GEO accession GSE329926
- Raw data including images, micrographs, and CSV files that go into these codes: Zenodo at DOI: 10.5281/zenodo.20033030
